Setup Instructions

1

Enable 2-Step Verification

Go to your Google Account → Security → 2-Step Verification → Turn On

2

Generate App Password

Google Account → Security → App Passwords → Generate for "Mail"

3

Set Environment Variables

$env:SMTP_SENDER_EMAIL = "your@gmail.com"
$env:SMTP_SENDER_PASSWORD = "your_app_password"
$env:SMTP_SENDER_NAME = "Your Name"
4

Restart the Server

Run python app.py again to apply the new settings.
